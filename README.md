# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer with instant results and a simple UI.

---

## 🚀 Features

- 🤖 Play against computer (random choices)
- ⚡ Instant result (Win / Lose / Draw)
- 📊 Score tracking
- 🔄 Play again functionality
- 📱 Responsive design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
